# contaspagar - core db

Referente a banco de dados utilizado na aplicação de contas a pagar.

Imagem docker
Flyway
Scripts sql de carga e estrutura


## Pré-requisito

Java 11
Docker

## Execução

Na pasta postgresql/base executar o scripts shell de build e run, nesta ordem.

Ao final do processo, se tudo der certo, terá um banco postgresql respondendo na seguinte url connection

localhost:5432/rmcdb
